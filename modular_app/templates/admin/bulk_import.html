üìã How to Use Bulk Import

  1. Download Template: Click "Download Template" button to get the Excel file with proper format and sample data.
  2. Fill Your Data: Open the Excel file, remove the sample row, and add your data. Follow the instructions in the template.
  3. Upload File: Click "Choose Excel File" and select your filled template.
  4. Import: Click "Upload & Import" and wait for the process to complete.
  5. Review Results: You'll see success count and any errors that occurred.

‚ö° Pro Tips

  • Start with a small batch (10-20 rows) to test the format.
  • For inventory, categories and groups are auto-created if they don't exist.
  • For employees, sites are auto-created if the site name doesn't exist.
  • Phone numbers must be 10 digits without country code.
  • If an employee/customer with the same phone exists, it will be skipped.
  • If an inventory item with the same SKU exists, it will be skipped.
